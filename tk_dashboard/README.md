# 🎯 TK Bar Detector PRO — Веб-дашборд

## 📊 Как пользоваться

### **1. Запусти бота:**
```bash
python tk_scanner.py --debug
```

### **2. Открой дашборд:**
Просто открой файл `tk_dashboard/index.html` в браузере (Chrome, Firefox, и т.д.)

**Или:**
```bash
# Windows
start tk_dashboard\index.html

# Mac
open tk_dashboard/index.html

# Linux
xdg-open tk_dashboard/index.html
```

### **3. Дашборд будет обновляться автоматически!**

- **Автообновление:** каждые 30 секунд
- **Кнопка "Обновить":** для ручного обновления

---

## 📈 Что показывает дашборд:

### **Основная статистика:**
- TK-пробоев за сегодня
- TK-ретестов (всего / #1 / #2 / #3)
- ✅ Успешных / ❌ Неудач / ⏳ Ждут
- 🎯 Процент успеха

### **По ретестам:**
- Статистика отдельно по #1, #2, #3
- Какой номер ретеста успешнее

### **По метрикам:**
- volume_ratio ≥ 2.0 vs < 2.0
- body_ratio ≥ 1.5 vs < 1.5
- Какие фильтры работают лучше

### **По направлению:**
- LONG vs SHORT статистика

### **Топ монет:**
- Лучшие монеты по % успеха
- Худшие монеты

### **Последние сигналы:**
- Таблица всех сигналов
- Результат каждого
- % движения цены

### **💡 Рекомендации:**
- Автоматические инсайты на основе статистики
- Какие фильтры применять

---

## 📁 Файлы:

| Файл | Что это |
|------|---------|
| `tk_dashboard/index.html` | Веб-дашборд |
| `tk_dashboard/stats.json` | Данные (обновляется ботом) |
| `tk_statistics.json` | Полная статистика (история) |

---

## 🔄 Как обновляются данные:

1. **Бот сканирует рынок** → новые сигналы
2. **Записывает в `tk_statistics.json`** → сохранение
3. **Обновляет `tk_dashboard/stats.json`** → данные для веба
4. **Дашборд читает `stats.json`** → показывает статистику
5. **Автообновление каждые 30 сек** → свежие данные

---

## 💡 Советы:

- **Оставь дашборд открытым** — он сам обновляется
- **Смотри статистику после 20+ сигналов** — больше данных = точнее выводы
- **Обращай внимание на #2 и #3** — они могут быть успешнее #1
- **Фильтруй по volume_ratio** — сигналы с высоким объёмом качественнее

---

## 🎨 Кастомизация:

Можешь изменить цвета в `index.html` (секция `<style>`):
- `background` — фон страницы
- `.card` — фон карточек
- `.success` — цвет успеха (зелёный)
- `.fail` — цвет неудачи (красный)
